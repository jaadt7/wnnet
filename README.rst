Overview
========

wnnet is a python package for handling
`webnucleo <http://webnucleo.org/>`_ reaction networks.

Installation
------------

Install from `PyPI <https://pypi.org/project/wnnet>`_ with pip by
typing in your favorite terminal::

    $ pip install wnnet
